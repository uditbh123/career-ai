# Career AI

Career AI is an open-source AI-powered career platform that helps users discover jobs, optimize resumes, generate tailored cover letters, prepare for interviews, and track career growth.

## Vision

Build an AI-powered career operating system that supports job seekers across multiple countries, starting with Finland and expanding throughout Europe.

## Status

🚧 Currently in active development.

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- Next.js
- React
- OpenAI / Anthropic / Gemini
