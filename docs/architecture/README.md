# Architecture Documentation

This directory contains the architectural design and technical decisions for the Career AI platform.

The goal of these documents is to explain how the system is designed, why specific technologies were chosen, and how different parts of the application interact with one another.

## Documents

* **system-overview.md** – High-level overview of the entire platform.
* **backend-architecture.md** – Backend design, layers, and responsibilities.
* **folder-structure.md** – Explanation of the project organization.
* **technology-stack.md** – Technologies used and the reasoning behind each choice.

These documents will evolve as the project grows.
